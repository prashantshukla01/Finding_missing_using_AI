# Lost Persons Database

This directory contains images of missing persons for face recognition.

## File Naming Convention
- Each image should be named as: `{person_name}.jpg`
- Example: `Aarav.jpg`, `Priya_Sharma.jpg`

## Requirements
- Face should be clearly visible
- Good lighting and resolution
- Front-facing or near-front-facing pose preferred
- Only one face per image

## Supported Formats
- JPEG (.jpg, .jpeg)
- PNG (.png)
- Maximum file size: 10MB
